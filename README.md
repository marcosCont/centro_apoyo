# Centro de Apoyo Academico

Tarea 3 de Paradigmas de Programacion. Sistema hecho en Prolog
que maneja una base de estudiantes con su curso y nota, permite consultar
su estado academico y tiene una calculadora, todo desde un menu.

## Que hace

- Guarda 30 estudiantes con nombre, curso y nota.
- Revisa si un estudiante pertenece a un curso.
- Dice si esta aprobado (nota >= 4.0) o reprobado (nota < 4.0).
- Pide un nombre por consola y muestra su nota y estado.
- Calculadora con suma, resta, multiplicacion y division.
- Menu que se repite hasta elegir salir.

## Como correrlo

Abrir el archivo en SWI-Prolog y cargarlo:

```prolog
?- [centro_apoyo].
```

Para usar el menu:

```prolog
?- menu.
```
